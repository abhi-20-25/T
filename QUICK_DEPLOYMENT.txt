🚀 INITIAL DEPLOYMENT (First Time Setup)

1. CLONE REPOSITORY:
   cd /home/ubuntu
   git clone https://github.com/abhi-20-25/Teatoast-Sakshi.git
   cd Teatoast-Sakshi

2. CREATE CAMERA CONFIG:
   nano config/rtsp_links.txt
   # Add: CameraName,rtsp://user:pass@ip:port/stream

3. INSTALL DEPENDENCIES:
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

4. START WITH DOCKER:
   chmod +x docker-start.sh
   ./docker-start.sh

5. ACCESS APPLICATIONS:
   Person Detection: http://server-ip:5000
   Queue Monitor:    http://server-ip:5001
   Occupancy Monitor: http://server-ip:5002

═══════════════════════════════════════════════════════════════════════

🔄 UPDATING CODE (Pull Latest Changes)

   cd /home/ubuntu/Teatoast-Sakshi
   ./docker-stop.sh
   git pull origin main
   docker-compose build
   ./docker-start.sh
